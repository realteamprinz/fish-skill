## What is fish.skill?

A species-specific extension of paw.skill that adds deep aquatic intelligence — tank ecosystem management, individual fish profiling from photos, social dynamics mapping, visual health monitoring, and tank evolution tracking.

paw.skill knows your pet. fish.skill knows your FISH.

## Install

Requires [paw.skill](https://github.com/realteamprinz/paw-skill) as parent.

```bash
# OpenClaw
clawhub install realteamprinz/paw
clawhub install realteamprinz/fish

# Hermes
hermes skills install paw-skill
hermes skills install fish-skill

# Claude Code
claude install-skill https://github.com/realteamprinz/paw-skill
claude install-skill https://github.com/realteamprinz/fish-skill
```
